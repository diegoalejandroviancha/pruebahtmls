TICS

son el conjunto de herramientas, recursos y sistemas tecnológicos que permiten el acceso, la gestión, el procesamiento, el almacenamiento y la transmisión de información de forma digital

LAS TIC

WORD

Word es un software de procesamiento de texto desarrollado por Microsoft que permite crear, editar, dar formato, guardar e imprimir documentos digitales de forma eficiente. Es ampliamente utilizado en entornos académicos, laborales y personales, ya que ofrece herramientas intuitivas para escribir, insertar imágenes, tablas, gráficos y aplicar estilos que mejoran la presentación del contenido. Además, facilita la revisión colaborativa y el control de cambios, convirtiéndose en una herramienta fundamental para la redacción y organización de información escrita.

EXCEL

Excel es un programa de hojas de cálculo desarrollado por Microsoft que permite organizar, analizar y visualizar datos de manera eficiente. A través de filas y columnas, los usuarios pueden realizar operaciones matemáticas, estadísticas y financieras utilizando funciones y fórmulas integradas. Además, Excel permite la creación de gráficos, tablas dinámicas y automatización de tareas mediante macros, lo que lo convierte en una herramienta esencial para la gestión de información en ámbitos como la contabilidad, la administración, la educación y la investigación.

ACCESS

Access es un sistema de gestión de bases de datos desarrollado por Microsoft que permite crear, administrar y manipular bases de datos de manera sencilla y visual. Combina una interfaz gráfica fácil de usar con herramientas de programación, lo que permite a los usuarios almacenar grandes cantidades de información, establecer relaciones entre diferentes tablas, y generar formularios, consultas e informes personalizados. Es ampliamente utilizado en entornos empresariales y educativos para organizar datos de forma eficiente sin necesidad de conocimientos avanzados en programación.
